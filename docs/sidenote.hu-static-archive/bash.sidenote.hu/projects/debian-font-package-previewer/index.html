Debian font package previewer

This shell script creates preview images from Debian font packages. The script uses GNU ar to extract Debian packages, FontForge to generate sample images of TrueType and OpenType font files, and ImageMagick to merge multiple images into one file.

Invoke the script the following way:

  • ./debian-font-package-previewer.sh <directory which contains Debian font packages>

Notes:

  • if no directory was specified then the script will search for package files in the current working directory;
  • read & write permissions are necessary for the specified directory.
