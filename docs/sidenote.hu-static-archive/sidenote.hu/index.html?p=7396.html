#!/bin/bash / Shell Scripts

Általam írt shell (Bash) scriptek gyűjtőoldala, melyek főként oktatási célokat, illetve saját szórakoztatásomat szolgálják. Az itt fellelhető shell scriptek mindegyike Bash ≥ 4.2-vel volt tesztelve, ezért a futtatásukhoz is ez ajánlott. A szkriptek GPLv3+ alatt vannak licencelve. Jelenleg a következő scriptek érhetők el:

  • ASCII labirintus generátor
  • Bash Gorillas (bash-gorillas)
  • Debian font csomag preview script
  • Game of Life (bash-life)
  • időjárás-jelentés (bash-weather)
  • logaritmikus spirál kirajzoló script
  • Mandelbrot halmaz kirajzoló script

  
  

Jó szórakozást!
